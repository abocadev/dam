Desarrollo de Aplicaciones Web

El Ciclo Superior Desarrollo de Aplicaciones Web o también denominado DAW es una titulación oficial de FP que se enfoca en el desarrollo, mantenimiento e implantación de aplicaciones informáticas en diferentes entornos web, tanto en internet como en intranet y extranet.

Los módulos que componen este Ciclo son los siguientes y podrás matricularte individualmente de cada uno de ellos:

Módulo: Nombre: Curso: Horas Semanales
Módulo 1 Sistemas informáticos Primero 6h
Módulo 2 Bases de datos Primero 6h
Módulo 2 Bases de datos (objetos/relaciones) Segundo 1h
Módulo 3 Programación Primero 5h
Módulo 3 Programación (Orientada a Objetos) Segundo 4h
Módulo 4 Lenguajes de Marcas y gestión de información Primero 3h
Módulo 5 Entornos de desarrollo Segundo 2h
Módulo 6 Desarrollo web en el entorno del cliente Segundo 5h
Módulo 7 Desarrollo web en el entorno servidor Segundo 5h

Módulo 10 Formación y Orientación Laboral Primero 3h
